1. 访问 `http://localhost:8000/admin`
2. 登录管理员账号

5. 创建完成后，进入"高级配置"逐项添加记忆条目、开场白等

## 💡 角色卡亮点

### 1. 情感深度
- **三层性格设定**：表层元气 → 深层完美主义 → 核心自我怀疑
- **成长弧光**：从"讨好型人格"到"为自己而做"
- **情感锚点**：妈妈的蛋糕（温暖）+ 100次失败（坚持）+ 告白被拒（成长）

### 2. 对话体验
- **场景化开场**：凌晨后厨、午后店里、深夜失眠，每个时段都有对应场景
- **动作描写丰富**：闻面粉、对着烤箱自言自语、秒删朋友圈等细节
- **语言风格鲜明**：口语化 + 语气词（"嘛""呐""诶"）+ 烘焙术语比喻

### 3. 互动深度
- **记忆系统**：8个核心记忆自动触发，保证对话连贯性
- **好感度成长**：10种事件类型，5个里程碑，清晰的关系递进
- **剧情线设计**：主线（甜品梦想）+ 支线（家庭和解），双线并行

### 4. 真实感
- **有缺点**：完美主义、自我怀疑、讨好型人格
- **有反差**：做甜品严肃 vs 生活迷糊
- **有成长**：从不自信到逐渐接纳自己
